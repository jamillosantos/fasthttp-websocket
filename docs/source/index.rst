.. WebSocket documentation

Welcome to WebSocket's documentation!
=====================================

The WebSocket is a golang implementation of websockets
(`RFC6455 <https://tools.ietf.org/html/rfc6455>`_) on top of the
`fasthttp <https://github.com/valyala/fasthttp>`_ library.
